Installation Instructions

  1. In Kodi: Settings → File Manager → Add Source
  2. Enter: https://pesicp.github.io/XStream-Player-Kodi21/
  3. Name it "XStream"
  4. Add-ons → Install from zip file → XStream → repository.xstream → Install
  5. Add-ons → Install from repository → XStream Repository → Video add-ons → XStream Player
